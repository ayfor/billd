The primary action control — square pixel corners, hard block shadow, presses "into the board" on click.

```jsx
<Button variant="primary" icon="plus">Add expense</Button>
<Button variant="secondary">Cancel</Button>
<Button variant="ghost" size="sm">Clear filters</Button>
<Button variant="primary" disabled>Saving…</Button>
```

Variants: `primary` (Sapphire fill), `secondary` (outlined card), `ghost`
(text-only). Sizes: `sm` | `md` | `lg`. Pass `icon` for a leading pixel icon
and `iconBase` to point at the icons folder relative to the page.
