The signature billd surface — a notched 2px pixel panel with a hard block shadow and an optional pixel-face eyebrow title.

```jsx
<Card title="RECENT EXPENSES">
  …rows…
</Card>

<Card raised={false} pad="var(--space-4)">flat inset panel</Card>
```

Cards sit on the same Shadow Grey as the canvas; the notched border and block
shadow carry the separation, not a lighter fill.
