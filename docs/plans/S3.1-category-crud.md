# S3.1 — Category CRUD with palette colors

> Story: [Notion S3.1](https://app.notion.com/p/37ca7227e26f81e7a60eeca1d33e26db) · Design: [F3 Categories](https://app.notion.com/p/37ca7227e26f81b99ee7e07c34635427) · Design system: [canonical](https://app.notion.com/p/37da7227e26f81f28e85fa5c6d1d38f8) · PR: #8 (base `main`)

## Objective

Build the Categories page (F3): create and edit categories with a palette-constrained colour picker, case-insensitive per-user unique names. (Usage stats + the strict delete guard are S3.2.)

## Data Model

No schema change — `Category` exists (S1.2) with `@@unique([userId, name])`. Colour is a palette **token name** string.

## Approach

1. **`src/lib/validation/category.ts`** — `categorySchema` (Zod): `name` 1–30 trimmed; `color` enum `["sapphire","seaweed","amethyst","lavender"]`. Export `CATEGORY_COLORS`.
2. **`src/app/(dashboard)/categories/actions.ts`** — `createCategory`, `updateCategory`. Each `requireUser()`; **case-insensitive uniqueness check** (`findFirst` name `equals` mode insensitive, excluding self on update) → "You already have that category"; user-scoped (`updateMany where id+userId`); `revalidatePath` for `/categories`, `/expenses`, `/dashboard` (name/colour propagate to pills + filters).
3. **`src/app/(dashboard)/categories/page.tsx`** (RSC) — `requireUser`, fetch categories (sortOrder asc), render header (title + count, Add button) + `<CategoriesClient>`.
4. **`src/components/CategoriesClient.tsx`** (client) — card grid (per F3: swatch + name, Edit link) + a create/edit modal (or inline form panel) with the **colour picker** (4 palette swatches; selected gets a 3px Lavender ring). Reuses `CategoryPill` style for swatches. Edit prefills.
5. **`src/components/CategoryColorPicker.tsx`** — the 4-swatch palette picker (controlled).

## Field rules (Design F3)

| Field | Type | Limits | Validation | Default |
|-------|------|--------|------------|---------|
| Name | text | 1–30, trimmed | required, unique per user (case-insensitive) | — |
| Colour | enum token | sapphire · seaweed · amethyst · lavender | required, one of palette | sapphire |
