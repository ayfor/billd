# S5.2 — Spend vs projection chart

> Story: [Notion S5.2](https://app.notion.com/p/37ca7227e26f816ebb77d0e7ef09da5c) · Design: [F5 Dashboard](https://app.notion.com/p/37ca7227e26f81b6bc13d40581dec33e) · Design system: [canonical](https://app.notion.com/p/37da7227e26f81f28e85fa5c6d1d38f8) · PR: #13 (base `main`)

## Objective

Add the dashboard's cumulative spend-vs-projection chart (Recharts): stepped Sapphire cumulative actual, dashed Amethyst projection to month end with the projected value at the endpoint, dashed expected-spend reference line, faint gridlines, today marker. (Adopted over daily bars per the Claude Design divergence review.)

## Data Model

No schema change.

## Approach

1. **`src/lib/queries/spendSeries.ts`** —
   - `dailyCumulative(userId, now)`: expenses in the current month grouped by day → running cumulative `{ day:number, cents:number }[]` for days 1..today.
   - `projectMonthEnd(spentCents, now)`: **pure** linear-pace projection = `round(spentCents / daysElapsed * daysInMonth)`; exported and **shared with S5.3's projection panel** (single source of truth for the projected number). Unit-tested.
2. **`src/components/SpendChart.tsx`** (client — Recharts requires it): `LineChart` with
   - actual series (Sapphire, `type="stepAfter"`) for days 1..today;
   - projection series (Amethyst, dashed) from today → last day at `projectMonthEnd`;
   - `ReferenceLine` y = expected (dashed Lavender) + label;
   - `ReferenceLine` x = today;
   - flat fills, square dots off, `CartesianGrid` faint Lavender. Receives server-computed data as props (no client fetching).
3. **Dashboard page** — fetch the series + expected (already have expected from S5.1's summary) and render `<SpendChart>` in the mid row. Empty month → chart empty-state strip.
4. Install `recharts`.

## Chart contract (Design F5)

| Element | Spec |
|---|---|
| Actual | cumulative running sum, stepped, Sapphire |
| Projection | today → month end, dashed Amethyst, endpoint = `projectMonthEnd` |
| Expected | horizontal dashed reference at Σ monthly + Σ yearly/12 |
| Today | vertical marker at current day |
| Grid | faint Lavender; no gradient fills, no data shadows |
