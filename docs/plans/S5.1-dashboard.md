# S5.1 — Dashboard shell, hero & recent expenses

> Story: [Notion S5.1](https://app.notion.com/p/37ca7227e26f81fbb764ec38c96e17fc) · Design: [F5 Dashboard](https://app.notion.com/p/37ca7227e26f81b6bc13d40581dec33e) · Design system: [canonical](https://app.notion.com/p/37da7227e26f81f28e85fa5c6d1d38f8) · PR: #12 (base `main`)

## Objective

Build the dashboard landing surface: hero total-spent vs expected, compact budget tiles (reusing `budgetStatuses`), and recent expenses. (Charts = S5.2, projection numeral = S5.3.)

## Data Model

No schema change.

## Approach

1. **`src/lib/queries/dashboard.ts`** — `dashboardSummary(userId, now)`: total spent this month (Σ all expenses, current calendar month UTC); **expected** = Σ monthly budget amounts + Σ yearly budget amounts / 12; recent 5 expenses (date desc, createdAt desc) with category. Reuses `budgetStatuses` for the tiles.
2. **Replace the dashboard stub** (`src/app/(dashboard)/dashboard/page.tsx`) with the real page: header (title + month, ⌘K chip, Add expense via quick-add), hero panel (total spent Pixelify numeral + expected line + Sapphire segment bar of spent/expected), budget tiles (top 4 by ratio via `budgetStatuses`, compact `BudgetBar`; "View all" → /budgets when >4), recent expenses (5 rows, click → edit via the existing modal path).
3. **Components**: `DashboardHero`, `BudgetTiles` (reuse `BudgetBar`), `RecentExpenses` (rows open the edit modal — reuse `ExpenseModal`; wrap in a small client island like `ExpensesClient`). Hero numeral uses the pixel display face.
4. **Empty states**: no budgets → hero shows spent only + "Set budgets to unlock projections" → /budgets; no expenses this month → hero $0.00 + recent shows last month's tail with a divider label.

## Computation contract (Design F5)

| Value | Formula |
|---|---|
| Total spent | Σ expenses, current calendar month (UTC) |
| Expected | Σ monthly budgets + Σ yearly budgets / 12 |
| Budget tiles | top 4 by ratio — `budgetStatuses` (shared with /budgets) |
| Recent | latest 5 by date desc, createdAt desc |

(The projection numeral + chart are S5.3 / S5.2; this story renders spent-vs-expected only.)
